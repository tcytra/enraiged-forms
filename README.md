# Enraiged Forms

**Enraiged Forms provides form handling features for Laravel Enraiged projects.**


## Usage

Install the Enraiged Forms package to your Laravel project using composer:

```sh
composer require tcytra/enraiged-forms
```

Publish the Enraiged Forms assets to the local project:

```sh
php artisan vendor:publish --tag=enraiged-forms --ansi --force
```
